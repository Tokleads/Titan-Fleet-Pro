2. The Fake DVLA Integration (Critical)
Problem: Showing fake endorsement data is dangerous. A manager might let a disqualified driver on the road because your system said "Clear."
Fix:
Option A (Recommended): Remove the feature entirely from the UI until you have the official DVLA API integration (which requires employer registration).
Option B (Stopgap): Replace the data with a manual upload field. "Upload Licence Photo" → Store in document storage → Manager manually verifies. Label it "Manual Verification Required."

4. GPS iOS Limitation (Product Strategy)
Problem: PWAs cannot track background GPS on iOS reliably.
Decision: You have two choices.
Choice A (Fast): Be transparent. Update your marketing: "Best experienced on Android devices. iOS users please keep app open during shifts."
Choice B (Strategic): Wrap your PWA in a Capacitor or React Native shell. This allows you to use native background geolocation plugins while keeping 90% of your web codebase. Recommendation: Do this before public launch if GPS tracking is a core selling point.
